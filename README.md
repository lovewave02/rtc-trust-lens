# RTC Trust Lens

A WebRTC demo that visualizes media integrity and network trust signals during real-time sessions.

## Why this is portfolio-worthy
- Deep browser/network/media domain knowledge.
- Security + UX explanation in one product.
- Distinct from common chat/video clones.

## MVP scope
- 1:1 WebRTC call skeleton
- Show bitrate/jitter/packet loss timeline
- Integrity signal panel (baseline hash + warnings)

## Tech stack
- Frontend: TypeScript
- Realtime: WebRTC
- Signaling: minimal Node server
